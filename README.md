# 🚀 MiniServe: Lightweight Node.js File Server

### A minimal (expressjs) Node.js file server designed to share files over a local Wi-Fi network. It is a high-performance, memory-efficient, and easy to deploy on Linux (Debian) or Windows.

## HOW TO GET STARTED ?

- COPY PASTE THE FILES YOU WANT TO SHARE INTO THE "/public" directory
- RUN npm run start
